## Identity

The brand mark is an italic Fraunces ampersand in a tandoori-red medallion, echoing the "Hand-made in Wales / Cymru" stamp on the pouch.

**Logo files (SVG):**

- [`blaze-monogram.svg`](./blaze-monogram.svg) — primary mark
- [`blaze-monogram-reversed.svg`](./blaze-monogram-reversed.svg) — reversed (for dark backgrounds)
- [`favicon.svg`](./favicon.svg) — tuned for 16–32px
- [`blaze-logo-horizontal.svg`](./blaze-logo-horizontal.svg) — full logo lockup (use this 90% of the time)
- [`blaze-logo-horizontal-reversed.svg`](./blaze-logo-horizontal-reversed.svg) — full lockup, reversed

- [`blaze-wordmark.svg`](./blaze-wordmark.svg) — wordmark only (for places where the monogram is already on-panel)

| Ink | `oklch(18% 0.02 60)` | Primary type |

## Type stack

- **Display:** Fraunces (700 upright, 400 italic for the `&amp;` — SOFT 20/30, WONK 1)
- **Body:** Inter
- **Mono:** JetBrains Mono

## Pouch design

- 60g resealable flat pillow pouch (peg-hook hang-hole, tear-notch, zipper reseal), Chilli &amp; Honey palette (red Tandoori, gold Masala)
- Canonical: [`blaze-pouch-v7-final.html`](./blaze-pouch-v7-final.html)
- 10 alternative colour-ways in the `blaze-pouch-v7-pair-v**-*.html` files
- Picked direction: **v02 Chilli &amp; Honey**

## Status

Pre-launch. Copy and packaging are UK labelling-compliant. Nutrition values, EAN barcode, batch numbers, maker address, and certifications (HMC, SALSA) are flagged as placeholders to confirm before print.
