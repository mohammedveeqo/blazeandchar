We never built a standalone logo file — the wordmark lives inside the landing page and the pouch. Below is the current wordmark, the type stack, and the brand palette as they're defined across the live files.

Blaze&Char

Identity · Wordmark

The Blaze & Char wordmark

Set in Fraunces 700 with SOFT 30, WONK 1, opsz 144. The ampersand is italic Fraunces 500, SOFT 80, sized 112% — rendered in the tandoori accent (pomegranate red).

Bookmark this as the reference — the wordmark isn't a separate SVG file, it's a live Fraunces render inside each page.

StockInkTandooriMasalaWalesPaper

Identity · Palette

The brand colours

Six tokens across all files. Stock is the warm tan pouch body; Ink is the primary type colour; Tandoori (red) and Masala (gold) are the SKU accents; Wales green tints the country mark; Paper is the landing-page background.

Defined in OKLch in every :root block.

03

Pouch master — the canonical artwork

The single source of truth for the pouch design. Every pair variant and colour-way below is generated from this file by overriding palette tokens — structure, copy, legal, and layout all flow from here.

Pouch · Canonical master (v7 final)

Tandoori + Masala — front & back

50g stand-up pouch, warm tan stock. Front: wordmark, hero name, BEEF JERKY category line, window, nutrition creds row, heat scale, three certs (hand-cut / 30g protein / Wales). Back: story, ingredients, nutrition panel, storage, batch + best-before, maker block, barcode placeholder, halal footer.

UK-labelling compliant (mustard allergen emphasis, topside not silverside, "born/reared/slaughtered UK" provenance language). The Wales silhouette is filled solid green via your clean single-path SVG.
